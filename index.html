Portfolio

I have included six of my 30 repositories on Github. The included projects include Github profile checker, Javascript clock, Rotating navigation bar,
Pricing panel, Double-click heart and Drink water

Github profile checker

This project makes use of Axios to fetch api data from Github. I used string concactination to link the input value to the api endpoint obtained from Github. I wrapped the whole async function in a try and catch block, it returns a user matching the input value and returns an error if the input value does not match any user name on Github.

Rotating Navigation bar

This project is CSS heavy, it uses the positioning properties to place the buttons in relayion to the web page. Javascript is used to toggle the rotation property and enables the page to slide to the right and also display the hidden navigation.

Theme Clock

I used a wide range of CSS properties to position the clock sticks and some in-built javascript properties such as .getMonth() method. And i also used javascript to toggle between Light and Dark Modes.

Double-click heart

This is a very simple project inspired by Instagram. The project is almost zero html, i accessed some in-built javascript methods such as element's offset values to get the position of each click on the image. On double-tapping, it displays a heart sign which fades away after a second and also updates how many times the image has been liked.

Pricing panel

This is one of the projects from Frontendmentor.io. It is a simple pricing panel found on many landing pages. I designed the toggle button and used javascript to change the pricing depending on what pricing plan is being toggled. Also some CSS to style each panel when hovered.

Drink water

One of my favourite projects. This project uses javascript to update the big cup once any of the small cups is clicked and update the the amount of cups left to complete two litres of the big cup. Its a fun little app that tracks water intake
